Features

* Display list of your FB pages in your wordPress plugin
* Post automatically your website contents to your FB pages
* Post randomisly old posts in a scheduled time
* Configure your auto-posts programs with category, date and numbers of posts 
* Set your own timezone
* Create multiple auto post programs
* Each auto posted article details is saved in a log file

== Installation ==

The following are the steps to install the FB Auto Post plugin

1. In your WordPress Administration Panels, click on Add New option under Plugins from the menu.
2. Click on upload at the top.
3. Browse the location and select the Social Media Auto Post and Scheduling contents Plugin and click install now.
4. To turn the Plugin on, click Activate.
5. Check your Administration Panels or WordPress blog to see if the Plugin is working.
6. You can change the plugin options from FB Auto Post under settings menu.
